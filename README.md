### Sobre as cores

Já existem cores pré-definidas no arquivo [`custom.css`](assets/css/custom.css), como variáveis. Tente ao máximo usar apenas essas cores, pois fazem parte da identidade visual do evento.

```css
:root { 
    --laranja: #FF5C27;
    --bege: #F5F3E7;
    --azul: #3F9BFF;
    --verde: #2A9507;
    --amarelo: #FFDA1F;
    --verde_escuro: #383806;
    --vermelho: #FD4000;
    --branco_default: #FFF;
    --preto_default: #000;
}
```
